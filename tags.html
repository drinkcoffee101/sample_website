Important HTML tags

This page is designed as a quick refrence for some of the common tags covered in Learn Enough to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header a container for introductory content or a set of navigational links
div division defines a division or a section in an HTML document
ol ordered list creates a numbered list
ul unordered list creates a bulleted list

Inline Elements

Tag Name Purpose Exmple Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make text strong <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough </a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Micheal Hartl">
code code make text code font <code>stuff n' stuff</code> stuff n' stuff
span span used to group inline-elements in a document <p>My mother has <span style="color:blue"> blue</span> eyes.</p>

My mother has blue eyes
li list item represents an item in a list <ol> <li>dog</li> </ol>
  1. Dog

Document Tags

Tag Name Purpose
html Hyper Text Markup Language Defines the document as a webpage. Also identifies the beginning and end of the HTML document.
head header Information about the document (i.e. Author, Title, which stylesheet to use, and meta tags) that will not appear on the actual page.
title title Defines the title that will appear in the title bar of your web browser. Must appear between the head tags.
body body Contains all the information and other visible content on the page
meta meta Contains information about the page that does not need to be displayed, but it still used by search engines and other web crawlers. They are typically used to give search engines a brief description of the page, as well as the important keywords on it.
